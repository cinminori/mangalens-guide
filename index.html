安裝步驟

  1. 用手機瀏覽器打開上方下載連結，點右上角下載圖示存到本機。
  2. 打開「檔案」App，找到剛剛下載的 MangaLens-v1.8.0.apk，點開。

覆蓋舊版裝不上：如果你裝過更早的版本，新版可能因為簽章不同無法直接覆蓋。

2註冊 Groq 帳號並取得 API Key

Groq 是目前 免費額度最佛心、速度也最快的 LLM 服務商，支援 Llama / Qwen / GPT-OSS 等開源模型，最適合拿來跑漫畫翻譯。

2-1. 註冊帳號

  1. 用電腦或手機瀏覽器打開 https://console.groq.com
  2. 右上角點「Sign Up」
  3. 選「Continue with Google」用 Google 帳號登入（最快）
    — 或用 Email 註冊也可以。
  4. 第一次登入會跳一些歡迎畫面，一路 Next / Skip 過就好。

2-2. 建立 API Key

  1. 登入後，左側選單點「API Keys」（也可以直接打開 console.groq.com/keys）
  2. 點右上角藍色按鈕「+ Create API Key」
  3. 命名隨便，建議寫「MangaLens」方便日後辨識
  4. 點「Submit」→ 會跳出 Key 內容，長得像 gsk_xxxxxxxxxxxxxxxx
  5. 點旁邊複製圖示，整串複製起來（建議先貼到記事本，因為這個視窗關掉就再也看不到了）
重要：Groq 的 API Key 只會顯示一次，視窗關掉後就再也看不到完整 key。如果搞丟了只能刪掉重建一支。
免費額度：Groq 對所有人提供慷慨的免費額度（每日重置），個人用翻譯漫畫完全用不完。不需要綁信用卡。

3在 App 內新增 OpenAI Profile

v1.8.0 的新架構：把 baseUrl + apiKey 集中在「OpenAI Profile」管理，5 個功能共享同一份設定，換 key 改一次就好。

3-1. 打開設定

  1. 打開 MangaLens
  2. 右上角點齒輪 ⚙ 進入「設定」
  3. 找到「OpenAI Profiles」分頁（或翻到設定最下方）

3-2. 新增 Groq Profile

  1. 點「+ 新增 Profile」
  2. 填入：
    • 名稱：Groq（隨便，看得懂就好）
    • Base URL：已預填 https://api.groq.com/openai（不用改）
    • API Key：把剛剛從 Groq 複製的 gsk_xxxx... 整串貼進去
  3. 點「儲存」
  4. 儲存後可以點「測試連線」，成功的話會跳出可用模型清單（看到一堆 llama / qwen / openai/gpt-oss 等等就對了）
小提醒：如果你之後想再加 OpenAI 官方、DeepSeek、Together AI 等其他兼容服務，重複以上步驟新增另一個 Profile 就好，可以共存。

4把 Profile 套到各功能

有了 Profile 後要決定每個功能用哪一個 Profile + 哪一個模型。建議第一次就先把翻譯引擎設好，其他可以之後再調。

4-1. 設定主翻譯引擎（Groq Vision + LLM）

這是最推薦的設定 — Vision 模型負責看圖 + OCR，後端 LLM 負責翻譯，兩段串起來。

  1. 進入「設定」
  2. 來源語言：依漫畫選「日文」或「韓文」
  3. 文字方向：日漫直書多選「直書」、網漫橫向就選「橫書」
  4. 翻譯引擎選「Groq Vision + LLM」
  5. 展開後會出現「GROQ VISION OCR」區塊：
    • Groq API Key：把剛剛複製的 gsk_xxxx... 整串貼進去
    • Vision 模型：填 meta-llama/llama-4-scout-17b-16e-instruct
  6. 往下找「翻譯後端」區塊：
    • 引擎選「OpenAI 相容」
    • Provider 下拉選「Groq」（如果還沒有就點旁邊齒輪去 OpenAI Profiles 新增）
    • 模型選 qwen/qwen3-32b 或 llama-3.3-70b-versatile
  7. 頁面最底按「載入模型清單」確認設定有讀到
e-ink 用戶：最上面有「大字介面」開關，打開後字級和工具列圖示會放大，Supernote / Boox 等觸控有點難精準的裝置很好用。

4-2. 各功能 Profile 對應表

主翻譯引擎（Vision）Groqllama-4-scout-17b-16e-instruct
後端翻譯模型Groqqwen/qwen3-32b

為什麼分視覺 vs 後端？視覺模型負責「看圖 + OCR」，需要多模態能力，llama-4-scout 是 Groq 上 vision 表現最好的。後端翻譯只處理純文字，qwen3-32b 中日韓翻譯品質很穩，llama-3.3-70b 則是穩定通用首選 — 兩個擇一即可，看你比較喜歡哪家的譯文風格。

5正式使用 — 框選翻譯漫畫

5-1. 開啟漫畫

MangaLens 有三種讀漫畫的方式：

  • 瀏覽器模式：直接在 App 內的瀏覽器開漫畫網站（推特、Pixiv、Kakao Webtoon 等）
  • 本機檔案：開 EPUB / PDF 漫畫
  • 系統分享：從其他 App 分享圖片到 MangaLens

5-2. 框選翻譯（核心操作）

  1. 在漫畫頁面點右下角的橘色 FAB（剪裁圖示）
  2. 進入框選模式後，拖曳框出要翻譯的對話框／好幾格／整頁都行
  3. 放手後會跳出「辨識中…」提示，等 2-5 秒（看模型速度）
  4. 翻譯完成，結果會以氣泡浮在原圖上，疊在對應位置

5-3. 點氣泡開啟功能視窗

點任何一顆翻譯氣泡，會從底部滑出操作視窗：

  • 標題列：顯示翻譯結果（例：「真的好開心喔」）
  • 原文區：顯示 OCR 出來的原文，已用空白把詞拆好（例：「楽に　実に　しみだ」）
    • 點擊單字 → 查字義（會用辭典引擎查）
    • 長按單字 → 拆解該詞的字根／組成
  • 底部四個按鈕：
    • 📋 複製 — 把翻譯結果複製到剪貼簿
    • ✏️ 編輯 — 手動修改翻譯文字（修錯字、調語氣都行）
    • 🔄 重譯 — 對同一塊原文再翻一次（換個說法）
    • 🗑 刪除 — 清掉這顆氣泡
省 token 小技巧：盡量只框「對話框」而不是整頁。雖然全頁也能翻，但小範圍翻譯品質更好、速度更快、用量更少。

6進階：辭典查字義 + 生字本

6-1. 查字義

  1. 點翻譯氣泡開啟「功能視窗」（見 5-3）
  2. 在「原文」區，點擊任一單字 → 查字義
  3. 會跳出該字的讀音、字義、詞性、例句
  4. 進階：長按單字可以拆解該詞的字根組成

6-2. 加入生字本

  1. 查完字後在辭典視窗點「加入生字本」
  2. 可以分類、加標籤、寫筆記
  3. 主畫面點「生字本」可以隨時複習

6-3. 匯入術語表

如果你在追特定作品有自訂譯名（例如角色名、招式名），可以做一份術語表：

  1. 準備一個 CSV，欄位：原文,譯文,備註
  2. 設定 → 術語表 → 匯入 CSV
  3. 之後翻譯會優先採用你定的譯名

?常見問題

Q1：翻譯結果一直跳「請求失敗」

  • 檢查 Profile 的 API Key 有沒有貼錯（前後不能有空白）
  • 到 Groq Console 看 Key 還在不在、有沒有被停用
  • 選的模型可能下架了 — 進 OpenAI Profile 重新 fetch 模型清單，挑現有的

Q2：翻譯品質不好

  • 後端翻譯模型可在 qwen/qwen3-32b 和 llama-3.3-70b-versatile 之間切換比較
  • 框選範圍縮小，一次只翻一個對話框
  • 或考慮加錢用 Gemini 2.5 Pro / GPT-4o 系列

Q3：可以離線用嗎？

不行。所有翻譯引擎都需要連網呼叫 LLM API。但讀漫畫本身（不開翻譯）可以離線。

Q4：圖會被上傳嗎？隱私如何？

你框選的圖片區塊會傳到你設定的那家 LLM 服務商（例如 Groq）。MangaLens 本身不會收集或上傳你的資料 — 只在你選的 Profile 和該家服務商之間直接通訊。

Q5：免費用到爆 Groq 額度怎麼辦？

  • Groq 每日重置免費額度，隔天就回來了
  • 或新增第二個 Profile 接 Gemini AI Studio（也有免費額度）
  • 或加錢升級 Groq 付費方案
